# Mandalareando

Tool for drawing mandalas

## Roadmap

* Essential features
    * Selecting axis count
    * Previewing brush
    * Drawing in all segments
* Drawing interface
    * Circle
    * Arcs
* Tools
    * Brush
    * Color
* Saving Data
    * Data Format (still or animated)
    * Playing
* App Evolution

## Tech used
* webpack
* pug
* eslint + eslint-config-airbnb-base
* stylus
* Canvas2D
